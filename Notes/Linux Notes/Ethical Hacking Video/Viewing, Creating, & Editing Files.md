### echo
`echo "hello"` can output what you say 
`echo "hey" > hey.txt` Can also be used to create a **new** file 
`echo "hey again" >> hey.txt` double >> used to **append** to a file 
### touch 
`touch newFile` creates a new file 
### nano
`nano newFile`
text editor 
### gedit 
`gedit newFile`
